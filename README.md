In order to run this project you need:

- A Chrome browser

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:Ochiengsteven Math Magicians.git
```

Install the prerequisites this project using:

```sh
  cd Math Magicians
  npm install
```
